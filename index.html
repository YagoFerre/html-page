O que são clientes

(voltar)

Cliente no linguajar web é o que consumimos do lado do computador, agora mesmo, vendo este blog somos o cliente consumindo as informações que vem da internet e servidor, assim renderizando um site ou aplicativo em nosso computador ou celular.

O que são servers

(voltar)

Servers são servidores no qual podemos armazenar arquivos que futuramente podem retornar esses arquivos para a internet. Existem vários tipos de servidores como: Servidor de aplicações, Servidor de arquivos, Servidor local, Servidor de E-mail etc...

(voltar)

Linguagem de programação é uma linguagem escrita de uma forma especifica que nos permite criar um conjunto de instruções e regras para desenvolver uma aplicação e Software.

Existem várias linguagens de programação com sua característica especifica usada para diferentes paradigmas e soluções.

Como é a estrutura básica do HTML

(voltar)

A estrutura básica do HTML é feita por tags, essas tags são: "html", "head" e "body". Essas tags são o esqueleto do HTML, porém existem outras tags para estilizar e dar vida a aplicação web como: "h1", "strong", "div", "p", "u" etc..

Voce saberá mais clicando aqui como usar essas tags e para qual sua utilidade.

Listas ordenadas e não ordenadas

(voltar)

As listas ordenadas

é uma lista de itens, porém não são marcadas com números, podendo colocar itens fora de sequência. Aqui está um exemplo:
  • Banana
  • Maça
  • Uva

As listas não ordenadas

são listas em ordem númericas, fazendo assim agrupar itens em sequência e ordem. Aqui está um exemplo:
  1. Arroz
  2. Feijão
  3. Macarrão
